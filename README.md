# Social-Aware-K-Means-Clustering-Algorithm
此專案欲實現一論文 [A social-aware K means clustering algorithm for D2D multicast communication under SDN architecture](https://www.sciencedirect.com/science/article/abs/pii/S1434841121000078) 
中的 `Social Aware K Means Clustering Algorithm` 演算法

## 需求
Python >= 3.9

## 使用方式

執行以下指令
```
python3 main.py
```

## DEMO
分群演算法參數採用論文中給予之定義資料，其中User Equipment以(500,500)為圓心，半徑500的圓內隨機產生之亂數座標。
